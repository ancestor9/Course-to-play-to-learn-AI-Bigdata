
## 📊 AI 데이터 분야 직군 및 학습 로드맵 정리

## 1️⃣ AI 데이터 분야 직군 소개 (Overview)

### 🔎 데이터 분석가 (Data Analyst)
- 목적: 데이터 기반 의사결정 지원
- 주요 업무:
  - 데이터 정제 및 가공, 통계 분석 및 시각화, 대시보드 제작
- 주요 기술:
  - Python, Pandas, SQL, Tableau / Power BI, 기초 통계

---

### 🏗 데이터 엔지니어 (Data Engineer)
- 목적: 데이터 인프라 구축 및 관리
- 주요 업무:
  - 데이터 파이프라인 설계, ETL 구축, 대용량 데이터 처리
- 주요 기술:
  - SQL, Python, Spark / Hadoop, Cloud (AWS, GCP, Azure), Linux

---

### 🤖 머신러닝 엔지니어 (ML Engineer)
- 목적: 머신러닝 모델 개발 및 배포
- 주요 업무:
  - 모델 학습 및 최적화, 모델 서빙 및 API 개발, MLOps 구축
- 주요 기술:
  - Python, PyTorch / TensorFlow, FastAPI, Docker, Cloud

---

### 🧠 데이터 사이언티스트 (Data Scientist)
- 목적: 데이터 기반 문제 해결 및 예측 모델 개발
- 주요 업무:
  - 가설 설정 및 실험 설계, 통계 모델링, 머신러닝 모델 개발
- 주요 기술:
  - Python, 통계학, 확률론, 머신러닝, 데이터 스토리텔링

---

## 2️⃣ 기본 역량 (Foundation Skills)

### 💬 질문 잘 하는 법

- 가설을 세우고 검증 질문하기, 재현 가능한 질문 작성하기
